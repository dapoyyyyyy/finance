Cara Menghidupkan Asisten AI (FlowAI)

FlowAI adalah asisten pintar di Dashboard yang bisa membedah keuanganmu secara instan dan personal. Untuk memberinya "nyawa", kaitkan dengan OpenRouter (penyedia AI gratis).
✓ 100% Gratis✓ Tanpa Kartu Kredit✓ Cuma 1 Menit
Buka situs openrouter.ai di tab browser baru.
Klik Sign In di pojok kanan atas, lalu login dengan akun Google (cara tercepat).
Buka halaman kunci: klik menu Keys atau langsung ke openrouter.ai/settings/keys.
Klik Create Key, beri nama bebas (mis. "Kunci FinanceFlow"), tekan Create.
Muncul kode rahasia panjang (diawali sk-or-…). Blok seluruhnya lalu Copy.
Kembali ke sini, Paste ke kotak di bawah, klik Simpan.
Selesai! Kembali ke Dashboard — FlowAI siap membedah keuanganmu. 🎉

Sinkronisasi Google Drive

Supaya data laptop & HP sinkron: app membuat folder FinancialOS-Daffa baru di Google Drive-mu dan menyimpan data + struk di sana. Hubungkan dengan akun Google yang sama di setiap perangkat.
📁
Cara membuat Client ID (±3 menit, sekali saja):
1. Buka console.cloud.google.com → login Google → New Project (nama bebas).
2. Menu APIs & Services → Library → cari Google Drive API → Enable.
3. APIs & Services → OAuth consent screen → External → isi nama app & email → Save. Di bagian Test users, tambahkan email Google-mu.
4. Credentials → Create Credentials → OAuth client ID → type Web application → di Authorized JavaScript origins masukkan alamat situsmu (mis. https://username.github.io) → Create.
5. Salin Client ID-nya ke kotak di atas → klik Hubungkan Drive → pilih akun Google-mu → izinkan.
Ulangi hanya langkah "Hubungkan Drive" di perangkat lain (HP) — datanya otomatis sama.
