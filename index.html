Menghidupkan Claude AI (Financial Advisor)

Claude (buatan Anthropic) membedah keuanganmu langsung di Dashboard — analisis tajam, saran personal, paham konteks London & LPDP.
✓ Kualitas analisis terbaik✓ Kunci milikmu sendiriℹ Perlu kredit kecil (~$5, awet berbulan-bulan)
Buka console.anthropic.com → daftar/login (bisa pakai Google).
Isi kredit di menu Billing (minimal $5 — dengan model Haiku, satu pertanyaan hanya ±$0.003, jadi $5 ≈ ribuan pertanyaan).
Buka menu API Keys → Create Key → beri nama (mis. "FinancialOS") → Create.
Salin kunci yang diawali sk-ant-… (hanya muncul sekali).
Paste ke kotak di bawah → Simpan → tes dengan tombol Tes Koneksi.
Kembali ke Dashboard — Claude siap jadi penasihat keuanganmu. 🎉
🔒
Kunci disimpan di datamu sendiri (perangkat + Google Sheets-mu), dipakai langsung dari browser ke Anthropic. Jangan bagikan file backup ke orang lain.

Sinkronisasi Google Sheets ☁️

Data tersimpan sebagai spreadsheet rapi bernama FinancialOS-Daffa di Google-mu: laptop & HP otomatis sama, dan kamu bisa membukanya kapan saja di app Google Sheets. Struk tetap di-backup ke folder Drive.
📊 Buka di Google Sheets
📁
Cara membuat Client ID (±3 menit, sekali saja, gratis):
1. Buka console.cloud.google.com → login → New Project.
2. APIs & Services → Library → Enable Google Drive API dan Google Sheets API.
3. OAuth consent screen → External → isi nama & email → Save → tambahkan email-mu di Test users.
4. Credentials → Create Credentials → OAuth client ID → Web application → Authorized JavaScript origins: alamat situsmu (mis. https://username.github.io) → Create.
5. Salin Client ID ke kotak di atas → Hubungkan Google → pilih akun → Allow (jika muncul "unverified app": Advanced → Go to app → Allow).
Di HP: buka situs yang sama → langsung klik Hubungkan Google.
